# 🌐 浏览器缓存清理指南

## 立即清理浏览器缓存 (可释放0.5-5GB)

### Chrome 浏览器清理
```
快捷键方法：
1. 按 Ctrl + Shift + Delete
2. 时间范围选择："时间不限"
3. 勾选以下项目：
   ✅ 浏览记录
   ✅ Cookie 和其他网站数据
   ✅ 缓存的图片和文件
   ✅ 下载记录
   ✅ 自动填充表单数据
4. 点击"清除数据"

手动方法：
1. 点击右上角三个点 → 更多工具 → 清除浏览数据
2. 选择"高级"标签页
3. 时间范围："时间不限"
4. 勾选所有项目
5. 点击"清除数据"
```

### Edge 浏览器清理
```
快捷键方法：
1. 按 Ctrl + Shift + Delete
2. 时间范围选择："所有时间"
3. 勾选以下项目：
   ✅ 浏览历史记录
   ✅ Cookie 和其他站点数据
   ✅ 缓存的图像和文件
   ✅ 下载历史记录
   ✅ 自动填充表单数据
4. 点击"立即清除"

手动方法：
1. 点击右上角三个点 → 设置
2. 左侧选择"隐私、搜索和服务"
3. 点击"选择要清除的内容"
4. 选择时间范围和要清除的数据
5. 点击"立即清除"
```

### Firefox 浏览器清理
```
快捷键方法：
1. 按 Ctrl + Shift + Delete
2. 时间范围选择："全部"
3. 勾选以下项目：
   ✅ 浏览和下载历史记录
   ✅ Cookie
   ✅ 缓存
   ✅ 活动登录
   ✅ 表单和搜索历史记录
4. 点击"立即清除"

手动方法：
1. 点击右上角三条线 → 设置
2. 左侧选择"隐私与安全"
3. 在"Cookie 和网站数据"部分点击"清除数据"
4. 勾选要清除的项目
5. 点击"清除"
```

### Safari 浏览器清理 (Mac)
```
1. 菜单栏选择 Safari → 偏好设置
2. 点击"隐私"标签页
3. 点击"管理网站数据"
4. 点击"移除全部"
5. 确认删除
```

## 🔧 高级浏览器清理

### 清理浏览器配置文件夹 (更彻底)
```
Chrome 配置文件夹：
位置：C:\Users\[用户名]\AppData\Local\Google\Chrome\User Data
可删除：Default\Cache, Default\Code Cache

Edge 配置文件夹：
位置：C:\Users\[用户名]\AppData\Local\Microsoft\Edge\User Data
可删除：Default\Cache, Default\Code Cache

Firefox 配置文件夹：
位置：C:\Users\[用户名]\AppData\Local\Mozilla\Firefox\Profiles
可删除：cache2 文件夹
```

### 重置浏览器 (终极清理)
```
Chrome 重置：
1. 设置 → 高级 → 重置和清理
2. 点击"将设置还原为原始默认设置"
3. 点击"重置设置"

Edge 重置：
1. 设置 → 重置设置
2. 点击"将设置还原为默认值"
3. 点击"重置"

Firefox 重置：
1. 帮助 → 故障排除信息
2. 点击"刷新 Firefox"
3. 确认刷新
```

## 📊 预期清理效果

### 轻度使用 (0.5-1GB)
- 基本缓存和历史记录
- 适合偶尔上网的用户

### 中度使用 (1-3GB)
- 大量缓存文件
- 适合经常上网的用户

### 重度使用 (3-5GB+)
- 大量缓存、下载、扩展数据
- 适合长期未清理的用户

## ⚠️ 注意事项

### 清理后会丢失：
- 浏览历史记录
- 保存的密码 (如果选择清理)
- 自动填充数据
- 网站登录状态

### 建议清理前：
- 导出书签
- 记录重要密码
- 同步浏览器数据到云端
