# C盘清理详细操作指南

## 🚀 快速开始

### 方法1：使用我提供的脚本
1. **自动清理脚本**：双击运行 `c_drive_cleanup.bat`
2. **分析脚本**：右键 `Analyze-CDrive.ps1` → "使用PowerShell运行"

### 方法2：手动操作

## 📋 立即可执行的清理操作

### 1. 清理临时文件 (5分钟，可释放1-5GB)
```
步骤：
1. Win + R 键，输入 %temp%，按回车
2. Ctrl + A 全选，按Delete删除
3. Win + R 键，输入 temp，按回车
4. Ctrl + A 全选，按Delete删除
5. 清空回收站
```

### 2. 运行磁盘清理工具 (5分钟，可释放2-10GB)
```
步骤：
1. Win + R 键，输入 cleanmgr，按回车
2. 选择C盘
3. 勾选所有可清理项目
4. 点击"清理系统文件"
5. 再次勾选所有项目，包括"Windows更新清理"
6. 点击确定
```

### 3. 清理浏览器数据 (3分钟，可释放0.5-2GB)
```
Chrome：
1. 按 Ctrl + Shift + Delete
2. 选择"全部时间"
3. 勾选所有项目
4. 点击"清除数据"

Edge：
1. 按 Ctrl + Shift + Delete
2. 选择"全部时间"
3. 勾选所有项目
4. 点击"立即清除"
```

## 🔧 中级清理操作

### 4. 卸载不需要的程序 (15分钟，可释放5-50GB)
```
步骤：
1. 设置 → 应用 → 应用和功能
2. 按"大小"排序
3. 检查大型程序，卸载不需要的
4. 特别注意：
   - 游戏（通常很大）
   - 旧版本软件
   - 试用版软件
   - Adobe产品（如果不用）
```

### 5. 清理系统还原点 (5分钟，可释放5-20GB)
```
步骤：
1. 右键"此电脑" → 属性
2. 系统保护 → 配置
3. 删除 → 删除此驱动器的所有还原点
4. 或者保留最新的，删除旧的
```

### 6. 清理Windows更新文件 (10分钟，可释放2-10GB)
```
方法1 - 使用DISM命令：
1. 以管理员身份运行命令提示符
2. 输入：dism /online /cleanup-image /startcomponentcleanup
3. 等待完成

方法2 - 手动清理：
1. 停止Windows Update服务
2. 删除 C:\Windows\SoftwareDistribution\Download 文件夹内容
3. 重启Windows Update服务
```

## 📦 高级移动操作

### 7. 移动用户文件夹 (30分钟，可释放10-100GB)
```
桌面文件夹：
1. 右键桌面文件夹 → 属性 → 位置
2. 点击"移动"
3. 选择其他盘的位置（如D:\Desktop）
4. 点击确定，系统会自动移动文件

同样操作：文档、图片、视频、音乐、下载文件夹
```

### 8. 移动虚拟内存 (10分钟，可释放4-32GB)
```
步骤：
1. 右键"此电脑" → 属性 → 高级系统设置
2. 性能 → 设置 → 高级 → 虚拟内存 → 更改
3. 取消"自动管理所有驱动器的分页文件大小"
4. 选择C盘，选择"无分页文件"，点击设置
5. 选择其他盘（如D盘），选择"系统管理的大小"，点击设置
6. 确定，重启电脑
```

### 9. 禁用休眠功能 (1分钟，可释放4-32GB)
```
如果你不使用休眠功能：
1. 以管理员身份运行命令提示符
2. 输入：powercfg /hibernate off
3. 按回车，休眠文件会被删除
```

## 🛠️ 专业清理操作

### 10. 清理WinSxS文件夹 (谨慎操作)
```
步骤：
1. 以管理员身份运行命令提示符
2. 输入：dism /online /cleanup-image /startcomponentcleanup /resetbase
3. 等待完成（可能需要很长时间）
注意：此操作不可逆，会删除所有备份组件
```

### 11. 移动程序到其他盘
```
方法1 - 重新安装：
1. 卸载程序
2. 重新安装到其他盘

方法2 - 使用移动工具：
1. 下载Steam Mover或类似工具
2. 选择要移动的程序文件夹
3. 选择目标位置
4. 工具会创建符号链接
```

## ⚠️ 安全注意事项

### 绝对不要删除的文件夹：
- C:\Windows
- C:\Windows\System32
- C:\Program Files
- C:\Users\[用户名]\AppData\Roaming

### 操作前必做：
1. 备份重要数据
2. 创建系统还原点
3. 确保有Windows安装盘或恢复盘

### 如果出现问题：
1. 使用系统还原
2. 运行系统文件检查：sfc /scannow
3. 运行DISM修复：dism /online /cleanup-image /restorehealth
